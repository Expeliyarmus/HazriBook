# GitHub Pages Upload Guide

## Files to Upload (Drag all these at once):

1. Required Files:
   - build/ (folder with all contents)
   - .nojekyll (empty file, will be created)

2. After upload:
   - Add commit message: "Add production build for GitHub Pages"
   - Click "Commit changes"

## After Upload:
1. Go to repository Settings
2. Click "Pages" in the left sidebar
3. Under "Build and deployment":
   - Source: "Deploy from a branch"
   - Branch: "main" / "/(root)"
   - Click "Save"

Your app will be live at: https://[your-username].github.io/hazribook/
